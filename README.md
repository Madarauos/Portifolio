# 💼 Portfólio — Paulo Otávio Câmara Rojas

Site de portfólio profissional desenvolvido em Next.js, com foco em apresentação clara para recrutadores. Destaca projetos reais com contexto de problema, papel do desenvolvedor e impacto gerado.

---

## ✨ Páginas

- **Home** — resumo de 30 segundos, stack principal e disponibilidade
- **Sobre** — trajetória, formação e skills técnicas
- **Projetos** — cards estruturados com Problema / Meu papel / Impacto
- **Contato** — formulário funcional com envio via EmailJS

---

## 🛠️ Stack Tecnológica

| Camada | Tecnologia |
|--------|-----------|
| Framework | Next.js 15 + React 19 |
| Linguagem | TypeScript |
| Estilização | Tailwind CSS |
| Formulário/Email | EmailJS |
| Fontes | Poppins, DM Mono, Instrument Serif |
| Hospedagem | Vercel |

---

## 🎨 Design

- Tema escuro com acento roxo `#ae00ff`
- Tipografia: **Poppins** (headings) · **DM Mono** (código) · *Instrument Serif* (decorativo)
- Layout responsivo mobile-first

## 📚 Aprendizados

- Pensar o portfólio do ponto de vista do **recrutador**, estruturando cada projeto com contexto de problema, papel e impacto — em vez de só listar tecnologias
- Integrar o **EmailJS** diretamente no front-end para envio de formulário sem precisar de back-end próprio
- Aplicar um sistema de tipografia com **três fontes distintas** (Poppins, DM Mono, Instrument Serif) de forma coesa, usando cada uma no contexto certo
- Construir um layout dark com acento de cor único (`#ae00ff`) que fosse legível e profissional ao mesmo tempo
